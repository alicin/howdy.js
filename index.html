Howdy.js

Crossbrowser customizable growl-like notifications.

  • - supports all major browsers
  • - highly customizable
  • - total size is 1kb minified and gzipped.
  • - magically creates your notification element with the content and options you passed in.
  • - you can pass in any html element like h1, img, iframe etc. as your message

Options

A list of all the avaliable options

option value default info
duration integer 6000 Duration in milliseconds before the notification disappears. You can use 'Infinity' without quote marks to show the notification until the user clicks on close button.
message string none Message to be shown inside the notification. You can pass in any html code inside here. You may want to preformat your html using css if you are passing html as message
width integer 600 Width of the notification container either as integer or percentage.
position object {top: '50px', right: '50px'} Pass in the position you want notifications to appear as a javascript object. Accepts 'top', 'bottom', 'left' and 'right' as keys and you can also use percentages as values like '5%'
type string none To use the pre-formatted notification states, you can pass 'success', 'error', 'info' as type. you can easily change the css of this pre-formatted states from howdy.css
onFirstNoticeCreate function none If there are no notifications present, this function will be called after the first notification is created.
onNoticeCreate function none This function will be called after any notification is created.
onNoticeDestroy function none This function will be called after any notification is destroyed.
onAllNoticesDestroy function none This function will be called after the last notification on screen is destroyed.

Usage

  • - Include jquery and howdy.js (howdy.js must come after the jquery library)
  • - Include the howdy.css file and customize it as you like. For example if you want to use a button image instead of 'close', just set the text-indent to -9999px and place your image using css.
  • - If you want to pass in html as the message don't forget to pre format the elements you are passing in and change the "width" option accordingly. (passing an image 1000px wide won't fit in a container 600px wide. doing so would be silly.)
  • - Create an object with the options that you'd like to specify and pass it into $.howdy() as an argument. (You don't need to specify all of them)
  • - Voila!

var options = {
	duration: 6000,
	message: 'Howdy Folks',
	position: {top: 50, right: 50},
	width: '600',
	type: 'success',
	onFirstNoticeCreate: function(){ alert('first item created') },
	onNoticeCreate: function(){ alert('item created') },
	onNoticeDestroy: function(){ alert('item destroyed') },
	onAllNoticesDestroy: function(){ alert('all items destroyed') }
}

$.howdy(options);
